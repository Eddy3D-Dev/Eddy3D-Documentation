## ![Case Run Icon](../images/icons/Case_Run.png) Case Run

![Case Run Component](../images/components/Case_Run-crop.png)

Prepare and run a UMCF case.
 Eddy3D 1.0.0.827

#### Input
* ##### Case 
Case to prepare and/or run.
* ##### Prep 
Prepare meshing and case setup.
* ##### Run 
Run the simulation solver.
* ##### Parallel 
Run the case in parallel if enabled.

#### Output
* ##### Case
Updated case after prepare/run.
* ##### Logs
Latest execution logs.
