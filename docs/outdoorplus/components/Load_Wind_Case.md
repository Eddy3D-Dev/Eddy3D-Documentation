## ![](../images/icons/Load_Wind_Case.png) Load Wind Case

![](../images/components/Load_Wind_Case-crop.png)

Reference an existing wind case folder (mesh/ + case_NNN) for post-processing.

#### Input
* ##### Folder 
Path to the wind case folder that contains mesh/ and case_NNN folders.

#### Output
* ##### Case
Wind case reference; plug into the post-processing components' Case input.
* ##### Cases
Paths of the direction case folders found in the case.
