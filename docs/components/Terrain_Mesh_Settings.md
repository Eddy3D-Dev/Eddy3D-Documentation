# ![](/images/icons/Terrain_Mesh_Settings.png) Terrain Mesh Settings - [[source code]](https://github.com/Eddy3D-Dev/Eddy3D/search?q=%22Terrain%20Mesh%20Settings%22)

![](/images/components/Terrain_Mesh_Settings-crop.png)

Configure mesh settings for terrain and underground regions. OutdoorPlus

#### Input

| Name | Nickname | Description |
| ---- | -------- | ----------- |
| SrfLC |  | Number of layers in the terrain surface. Optional; default is 4. |
| SrfER |  | Expansion ratio of terrain surface. Optional; default is 1.2. |
| SoLC |  | Number of layers in the terrain soil. Optional; default is 4. |
| SoER |  | Expansion ratio of terrain soil. Optional; default is 1.2. |

#### Output

| Name | Nickname | Description |
| ---- | -------- | ----------- |
| Mesh |  | Mesh settings for terrain and underground regions. |
