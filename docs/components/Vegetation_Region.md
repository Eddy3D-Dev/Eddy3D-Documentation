# ![](/images/icons/Vegetation_Region.png) Vegetation Region - [[source code]](https://github.com/Eddy3D-Dev/Eddy3D/search?q=%22Vegetation%20Region%22)

![](/images/components/Vegetation_Region-crop.png)

Create a vegetation region with properties and mesh settings. OutdoorPlus

#### Input

| Name | Nickname | Description |
| ---- | -------- | ----------- |
| Mesh |  | Vegetation meshes for the region. |
| LAD |  | Leaf area density for vegetation. |
| Props |  | Optional vegetation property settings. |
| MeshSet |  | Optional meshing settings for vegetation. |

#### Output

| Name | Nickname | Description |
| ---- | -------- | ----------- |
| Vegetation |  | Vegetation region object for the case. |
