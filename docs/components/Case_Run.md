# ![](/images/icons/Case_Run.png) Case Run - [[source code]](https://github.com/Eddy3D-Dev/Eddy3D/search?q=%22Case%20Run%22)

![](/images/components/Case_Run-crop.png)

Prepare and run a UMF case. OutdoorPlus

#### Input

| Name | Nickname | Description |
| ---- | -------- | ----------- |
| Case |  | Case to prepare and/or run. |
| Prep |  | Prepare meshing and case setup. |
| Run |  | Run the simulation solver. |
| Parallel |  | Run the case in parallel if enabled. |

#### Output

| Name | Nickname | Description |
| ---- | -------- | ----------- |
| Case |  | Updated case after prepare/run. |
| Logs |  | Latest execution logs. |
