Read an OpenFOAM case directory and list its file containers.

#### Input
* ##### Dir 
Directory containing the OpenFOAM case.

#### Output
* ##### Files
File containers found in the case.
* ##### Ignored Files (I)
Names of files in the directory that were not added to the case.
