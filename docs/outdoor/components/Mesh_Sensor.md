![](../images/components/Mesh_Sensor-crop.png)

Mesh Sensor Grid  Converts a mesh surface into a grid of sensors for MRT analysis. Each mesh face serves as a measurement point for radiation and thermal comfort.   Eddy3D 0.5.0.815

#### Input
* ##### Mesh 
Analysis grid mesh. Each face center becomes a sensor point.

#### Output
* ##### Sen
Sensor points for MRT Simulation component
