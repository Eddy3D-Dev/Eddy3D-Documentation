## ![](../images/icons/SLURM_Runner.png) SLURM Runner

![](../images/components/SLURM_Runner-crop.png)

Add batch running files for SLURM  Eddy3D 0.5.0.815

#### Input
* ##### Res 
Eddy Result
* ##### CA 
Add the SLURM charge account as a string
* ##### NE 
Add the SLURM notification email as a string
* ##### MCPU 
Specify RAM allocated per CPU, e.g. 10 for 10G
* ##### JD 
Specify the duration of the SLURM job in hours, e.g. 10 for 10 hours
* ##### OFLD 
Specify the OpenFOAM load command as string, "module load openfoam-org/8-mva2-off3dy"

#### Output
