CO2 Source

 Simulates carbon dioxide generation, typically from occupants. Use this to assess ventilation effectiveness and air quality.

* ##### Geo 
Geometry

Name
* ##### IR 
Injection Rate imposed on object

Type: Absolute [-] or specific [1/m³]

Function Object
